HTML Star Setup Guide

1. Introduction

Welcome to the setup guide for HTML Star. This document will walk you through the process of setting up HTML Star, integrating it with My_Web_Console, and running your first program.

2. Prerequisites

  • Node.js (version 14.x or later)
  • npm (Node package manager)
  • Git

3. Setup Instructions

3.1 Clone the Repositories

git clone https://github.com/JoeySoprano420/HTML-Star.git
git clone https://github.com/JoeySoprano420/My_Web_Console.git

3.2 Install Dependencies for HTML Star

cd HTML-Star
npm install

3.3 Build HTML Star Project

npm run build

3.4 Integrate HTML Star with My_Web_Console

cd My_Web_Console
npm install
npm run build

Note: Ensure that you have all prerequisites installed before running the commands. If you encounter any issues, check the logs for errors and refer to the troubleshooting section.

3.5 Run the Setup Script

The setup script will automate the process of installing dependencies, building projects, and integrating HTML Star into My_Web_Console.

node setup.js

4. Writing and Running HTML Star Programs

4.1 Write Your First Program

Create a new file with the .htmlstar extension and add the following code:

+HTML Star+

~HTML Star~

<>
@function greet(name: String): Void
{
    @let greeting: String = "Hello, " + name + "!";
    @console.log(greeting);
}
*

<>
@greet("World");
*

+Finished+

4.2 Run the Program

Use the My_Web_Console IDE to load and execute your HTML Star program. You should see the output in the console.

5. Troubleshooting

If you encounter issues during setup or execution, consider the following steps:

  • Check the console output for error messages and follow any suggestions provided.
  • Ensure that all prerequisites are installed correctly.
  • Refer to the documentation for HTML Star and My_Web_Console for additional help.
